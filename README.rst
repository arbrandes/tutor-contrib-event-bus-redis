event-bus-redis plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/openedx/tutor-contrib-event-bus-redis

Usage
-----

::

    tutor plugins enable event-bus-redis
